================================================================================
                      TROUBLESHOOTING
================================================================================

Port conflicts?
  lsof -i :3000 or :8080
  kill -9 <PID>

Map not showing?
  Check MapView dynamic import with ssr: false
  Verify Suspense boundary

API 404 errors?
  Ensure backend running on :8080
  Check endpoint URLs

WebSocket issues?
  App works without WebSocket
  Check WS_URL environment variable

Build errors?
  rm -rf .next node_modules
  pnpm install && pnpm build
